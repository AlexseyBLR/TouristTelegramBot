Для запуска необхоимо запустить метод main класса TouristTelegramBotApplication.

База данных является удаленной, поэтому нет необходимости в файлах со скриптами для ее создания.

Т.к. база удаленная, то необходимо некоторое время для подключения к ней, иногда это может занимать до 1 мин.
При разработке использовалась база MySQL, там все работало моментально. Но для удобства проверки Вами, перенес базу на удаленной сервер.

Как было написано в условии, бот работает только на то, чтобы искать города в бд.
Так же предусмотрена возможность, когда есть два одноименных города. В этом случае в качестве ответа от бота, вы получите несколько кнопок, 
на которых будут указаны названия этих городов + страна в скобках. При нажатии на одну из кнопок будет подгружаться описание из бд к 
конкретному городу.
